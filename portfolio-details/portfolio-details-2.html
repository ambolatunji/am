Project Information

  • Title: Crime Analysis Using HCA
  • Category: Data Science
  • Client: Mr. Tobi Dare
  • Project Date: 17 August, 2020
  • Project URL: Crime Analysis Using HCA

Project Overview

This study involves analyzing the 2017 Nigeria crime dataset using the Hierarchical Clustering Algorithm (HCA). The data, sourced from the Nigeria Bureau of Statistics, was meticulously preprocessed to ensure accuracy before applying the clustering technique to identify states prone to crime.

Key Features

  • Data Preprocessing and Cleaning
  • Application of Hierarchical Clustering Algorithm (HCA)
  • Visualization of Crime Trends Across Various States
