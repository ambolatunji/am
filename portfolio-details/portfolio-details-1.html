Project Information

  • Title: Development of University Admission Prediction System
  • Category: Machine Learning
  • Client: Mr. Adebisi Akanji Sodiq
  • Project Date: 10 June, 2018
  • Project URL: Admission Prediction System

Project Overview

This project involves developing a predictive system to assess the likelihood of a student being admitted into a university based on various factors. Utilizing machine learning algorithms, the system was designed to process historical data and provide a probability score for each applicant.

Key Features

  • Data pre-processing and feature selection.
  • Model training using decision trees and hyperparameter tuning.
  • Comprehensive evaluation of model accuracy on test datasets.
